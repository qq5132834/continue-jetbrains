```
1. 菜单栏 file -> open -> 打开1.0.2 版本 -> 生成一个 gradle 项目 （参考导入源代码.png）
   注意：不要通过 open 一个存在的项目的方式。
2. 打包
   Tasks -> intellij -> buildPlugin

3. 调试
   Run Configurations  -> Run Test IDE
```
